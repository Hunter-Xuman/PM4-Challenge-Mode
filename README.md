# PM4 Challenge Mode
PM4 Challenge Mode is a mod of Paper Mario Sticker Star that adds some slight changes to the game to create a more challenging experience.

# Features
- Enemies No Longer Disappear When Running Away
- (WIP) New Flashy and Big Enemy Variants Added
- (WIP) Overworld Sticker Placements Reduced
- (WIP) Enemy Sticker Drops Altered
